## 📌 Overview

LearnSphere is a front-end Learning Management System (LMS) designed to provide an interactive online learning experience. The platform allows students to browse courses, track progress, watch lessons, complete quizzes, and manage their learning journey.

The project is developed entirely using **HTML5, CSS3, and Vanilla JavaScript** without any frameworks or backend services.

## 🏃‍♂️ How to Run the Project

### Step 1

Download or clone the repository.

### Step 2

Open the project folder.

### Step 3

Open:

```text
index.html
```

in your browser.

### Using VS Code

1. Install VS Code.
2. Install the Live Server extension.
3. Open the project folder.
4. Right-click `index.html`.
5. Select **Open with Live Server**.
